使用默认的分页器

数据库内存的数据较多，读取的时候一页太长，因此需要使用分页器进行分页，在 Django 的 views 内声明。

这样，在 URL 上就可以通过 ?page=x 来访问第 x 页的数据，例如：http://127.0.0.1:8000/news.html?page=6 展示的就是第六页的数据。
